Le setup, étape par étape

Voici exactement ce qui se passe après votre commande — sans surprise.

✅ Étape 1 — Confirmation immédiate

Dès votre paiement, vous recevez un email de confirmation. L'équipe Avistars est notifiée et vous est assignée.

📞 Étape 2 — Appel de configuration (J+1, 15 min)

Un membre de l'équipe vous appelle dans les 24h. Lors de cet appel on :
• Relie votre fiche Google à votre espace Avistars
• Personnalise le message de filtrage (votre ton, votre identité)
• Configure l'email de réception des retours négatifs
• Répond à toutes vos questions

🎨 Étape 3 — Fabrication du QR code (J+1 à J+3)

Votre QR code premium est créé et testé. Vous recevez :
• Le fichier PDF haute résolution imprimable
• La version numérique (pour votre site, Instagram, etc.)
• 3 chevalets de table physiques (Paris intra-muros) ou envoi postal

🚀 Étape 4 — Go live (J+3 maximum)

Le système est actif. Dès que vos premiers clients scannent, le filtrage fonctionne automatiquement. Vous n'avez rien d'autre à faire.

📊 Étape 5 — Suivi hebdomadaire

Chaque semaine vous recevez un récapitulatif : scans, avis positifs collectés, retours négatifs interceptés, évolution de note. Vous voyez tout, en temps réel.

❓ Une question avant de se lancer ?
